# SenpaiChat

This repository contains the source for **Senpai Bot**, a Pyrogram-based Telegram bot with a modular plugin architecture. It includes features such as:

- `/start` onboarding with customizable media and messages
- Automatic user/group registration and broadcast system
- AI chat powered by the Groq API
- Truth-or-Dare game with scoring, leaderboards, and monthly resets
- Global and group ranking system with badges
- Deployment guides for local, Docker, VPS, and Heroku

See `DEPLOYMENT.md` for setup instructions and `senpai_bot/config.py` for configuration options.
